# DominDev DominDynamics

Portfolio i strona marki `DominDynamics` zbudowana w `React 19 + Vite + Tailwind CSS + Framer Motion`.

## Stack

- React 19
- Vite 6
- Tailwind CSS
- Framer Motion

## Uruchomienie

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Wynik buildu trafia do `dist/`.
